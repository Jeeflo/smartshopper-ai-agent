# smartshopper-ai-agent

AI Agent untuk toko online SmartShopper menggunakan Google ADK, MongoDB Atlas, dan RAG.

## Teknologi
- Google ADK
- MongoDB Atlas
- Sentence Transformers (RAG)
- Gemini 2.5 Flash

## Fitur
- Routing otomatis antara Product Recommendation dan Common Information
- RAG untuk menjawab pertanyaan umum dari database MongoDB
- AI Agent dengan 2 tools terintegrasi

## Cara Menjalankan
1. Buka file `SmartShopper_AI_Agent.ipynb` di Google Colab
2. Ganti `GEMINI_API_KEY` dengan API key kamu
3. Jalankan semua cell (Runtime → Run All)
